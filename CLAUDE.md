# CLAUDE.md — DragonForge Link Demo

## Project

Dragon Forge Link — a browser-based "Hold & Spin" slot game demo (Dragon Link style).
Single-page static build: the game runs entirely from [index.html](index.html), with art
assets in `imagine_images/`. No build step or package manager — open `index.html` in a
browser to run.

Reference: [DRAGONFORGE_LINK_HANDOFF.md](DRAGONFORGE_LINK_HANDOFF.md).
